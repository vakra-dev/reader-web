# reader-web

Website and playground for [Reader](https://github.com/vakra-dev/reader) - open-source web scraping for LLMs.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Playground API

The playground requires a Reader API backend. Set the following environment variables:

```bash
READER_API_URL=https://api.reader.dev/api/v1
READER_API_KEY=your_api_key_here
```

## Deployment

Deploy to Vercel:

```bash
vercel
```
